Below are some examples to explain services usage by using HTTP/GET and KVP encoding.
Basic WFS Requests
  • GetCapabilities
  • DescribeFeatureType
  • GetFeature
Request Description
GetCapabilities

Retrieve capabilities information

 ^ up
WFS 1.1.0 GetCapabilities

Retrieve capabilities information by using the following parameters.

HTTP KVP Request:
.../services?
request=GetCapabilities
version=1.1.1
service=WFS
DescribeFeatureType

Retrieve schema description of a FeatureType

 ^ up
WFS 1.1.0 DescribeFeatureType A DescribeFeatureType request, which results the WFS FeatureTypes.

HTTP KVP Request:
.../services?
service=WFS
version=1.1.0
request=DescribeFeatureType
GetFeature

Retrieve feature instances by featuretype name. Optional constraints may be added.

 ^ up
WFS 1.1.0 GetFeature StateBoundary

A GetFeature request, which retrieves all features of a type (example: StateBoundary)

HTTP KVP Request:
...services?
service=WFS
version=1.1.0
request=GetFeature
typename=app:StateBoundary
namespace=xmlns(app=http://www.deegree.org/app)
