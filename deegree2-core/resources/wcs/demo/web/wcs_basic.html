Below are some examples to explain services usage by using HTTP/GET and KVP encoding.
Basic WCS requests
  • GetCapabilities
  • DescribeCoverage
  • GetCoverage
Request / Description Result
GetCapabilities

Retrieve capabilities information

 ^ up
WCS 1.0.0 GetCapabilities

${default.online.resource}?request=GetCapabilities&version=1.0.0&service=WCS
WCS 1.0.0 GetCapabilities Service Section

GetCapabilities request to get the service section of the Capabilities Document

...services?
service=WCS
version=1.0.0
request=GetCapabilities
section=/WCS_Capabilities/Service
DescribeCoverage

describes coverages served by the WCS

 ^ up
WCS 1.0.0 DescribeCoverage

A DescribeCoverage request which describes all offered coverages:

...services?
service=WCS
version=1.0.0
request=DescribeCoverage
WCS 1.0.0 DescribeCoverage saltlakesatellite

This DescribeCoverage request describes the coverage saltlakesatellite:

...services?
service=WCS
version=1.0.0
request=DescribeCoverage
coverage=saltlakesatellite
GetCoverage

Puts out images of coverages

 ^ up
WCS 1.0.0 GetCoverage saltlakesatellite

A GetCoverage request which puts out an image of the saltlakesatellite coverage:

...services?
service=WCS
version=1.0.0
request=GetCoverage
coverage=saltlakesatellite
crs=EPSG:26912
BBOX=420015.601297584,4508009.18775396,431966.87709265726,4519960.463549033
width=500
height=500
format=jpeg
