catalogueManager Hinweise für Oracle

Wenn Sie eine Oracle Datenbank zur Speicherung ihrer Metadaten nutzen wollen, müssen Sie zuvor sicherstellen, dass der entsprechende JDBC Treiber sowie ggf. weitere erforderliche Java libraries in das Verzeichnis: WEB-INF/lib kopiert werden. Denn leider sind wir aufgrund der Lizenzbestimmungen von Oracle nicht berechtigt, diese libraries mit zu liefern.

Für Oracle 10g werden die Dateien: ojdbc14_10g.jar, sdoapi.jar und sdoutl.jar benötigt. Für Oracle 11g sind es die Dateien: ojdbc6_g.jar (nur bei JDK 6) oder ojdbc5_g.jar (bei JDK 5 oder JDK 6 ), sdoapi.jar und sdoutl.jar benötigt.

Sie finden die Dateien als Bestandteil ihrer Oracle Installation oder können Sie über die Oracle-Seiten beziehen.
