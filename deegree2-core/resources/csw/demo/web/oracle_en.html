catalogueManager Hints for Oracle

If you are using an Oracle database for storing your metadata you must ensure that Oracle JDBC driver and Oracle geometry libraries has been copied into directory WEB-INF/lib. That must be done by yourself because we are not allowed to redistribute these libraries.

For Oracle 10g ojdbc14_10g.jar, sdoapi.jar and sdoutl.jar are required. For Oracle 11g you must copy ojdbc6_g.jar (for JDK 6) oder ojdbc5_g.jar (for JDK 5 or JDK 6 ), sdoapi.jar and sdoutl.jar into your WEB-INF/lib directory.

You will find these file as part of your oracle installation or you can download them from Orcale web pages.
