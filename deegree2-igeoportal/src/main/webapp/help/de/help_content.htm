Über deegree

  Copyright by:

deegree ist eine freie Software Initiative, die von der Arbeitsgruppe GIS des Geographischen Instituts der Universität Bonn, und der Firma lat/lon gegründet wurde. Unterstützung bei der Weiterenticklung wird sehr begrüßt.

Das freie Software Projekt deegree bietet die derzeit umfangreichste Umsetzung von Bausteinen zum Aufbau einer Geodateninfrastruktur, in der die Standards von Open GIS Consortium (OGC) und ISO/TC 211 implementiert sind. Da die gesamte Architektur von deegree auf OGC Spezifikationen und Konzepten basiert, gibt es keine Probleme bei der Integration standardisierter Produkte anderer Anbieter (z.B. ArcIMS von ESRI(c)).

Alle der genannten Server-seitigen Dienste und das Client Framework werden vom Open Source Projekt deegree bereitgestellt. deegree ist freie Software, für dessen Nutzung keine Lizenzkosten anfallen. Es kann frei kopiert und verteilt werden, auch die Erweiterung des Systemumfangs ist nicht mit Lizenzkosten verbunden. Die Offenheit des Quellcodes unterstreicht die Kostensicherheit des Systems. Da deegree in Java implementiert ist, ist es plattformunabhängig und kann in heterogenen Netzwerken verwendet werden. Installationen sind auf Windows XP/2000/NT, LINUX, Macintosh OS X and Solaris Servers lauffähig.

deegree ist freie Software, die der GNU Lesser General Public License unterliegt.

Übersichtskarte

Lagedarstellung

In der übersichtskarte werden Position und Größe des aktuellen Kartenfensters bezogen auf die festgehaltene Darstellung des Raumausschnitts durch einen rechteckigen Rahmen angezeigt. Bei der Betrachtung sehr kleiner Kartenausschnitte (großmaßstäbige Darstellungen) wird der Rahmen durch ein Fadenkreuz ersetzt.

Das Kartenfenster kann durch Anklicken einer beliebigen Stelle der übersichtskarte zentrisch neu positioniert werden, wobei seine Größe nicht verändert wird.

Werkzeugleiste

Einige Werkzeuge lösen direkt eine Aktion aus oder öffnen einen Dialog für weitere Eingaben.
Die anderen Werkzeuge steuern die Funktion eines Mausklicks im Kartenfenster. Den jeweils aktiven Modus erkennt man an einer Farbänderung des zugehörigen Symbols.
Ein einmal ausgewählter Modus bleibt bestehen, bis ein anderes Modus-Werkzeug ausgewählt wird.

Karte neu zeichnen

Der aktuelle Kartenausschnitt wird neu gezeichnet.

Anfangsansicht

Die aktuell ausgewählte Karte wird in ihrem Anfangszustand geladen. Dazu wird das Kartenfenster wieder auf den Ausschnitt positioniert, mit dem die Karte ursprünglich geladen worden ist. Die Sichtbarkeit von Themen und die Zuordnung von Sachdatenabfragen werden auf die voreingestellten Werte zurückgesetzt. über den Dialog Themenliste erweitern hinzugeladene Themen werden aus der Themenliste entfernt.

Vorherige Ansicht

Ruft die aktuelle Kartenansicht mit der letzten Boundingbox der Historie auf

Nächste Ansicht

Ruft die aktuelle Kartenansicht mit der nächsten Boundingbox der Historie auf

Vergrößern

Ein Klicken an eine beliebige Stelle innerhalb des Kartenfensters führt in diesem Modus zu einer Zentrierung der Karte auf diese Position und einer Vergrößerung des Darstellungsmaßstabs um 25 % (z.B. von 1:5000 auf 1:3750).
Alternativ kann mit gedrückter linker Maustaste ein rechteckiger Bereich im Kartenfenster aufgezogen werden, der dann formatfüllend dargestellt wird.

Verkleinern

In diesem Modus bewirkt ein Klicken an eine beliebige Stelle innerhalb des Kartenfensters eine Zentrierung der Karte auf diese Position und eine Verkleinerung des Darstellungsmaßstabs um 25% (z.B. von 1:5000 auf 1:6250).

Zur Kartenebene zoomen (Zoom to layer)

Dieses Werkzeug zoomt zur Standard-BoundingBox des ausgewählten Themas.

Sachinformation abfragen

In diesem Modus löst ein Klick innerhalb des Kartenfensters eine Sachdatenabfrage aus, deren Ergebnis in einem eigenen Fenster angezeigt wird.

Die Sachdatenabfrage bezieht sich auf das aktuell bei der Zuordnung von Sachdatenabfragen ausgewählte Thema.

Neu zentrieren

Der Kartenausschnitt wird auf die Position, die im Kartenfenster angeklickt wird, zentriert.

Karte verschieben

In diesem Modus kann die Karte an einem beliebigen Punkt angeklickt und mit gedrückter linker Maustaste verschoben werden. Durch Loslassen der Maustaste wird die Verschiebung abgeschlossen, die fehlenden Teile des Kartenbildes werden automatisch nachgeladen. Der Darstellungsmaßstab wird dabei nicht verändert.

Themenliste / Themenbaum erweitern

Mit diesem Werkzeug wird ein Dialog geöffnet, der die Möglichkeit bietet, weitere im Internet verfügbare Kartenangebote hinzuzuladen. In dem Dialog ausgewählte Themen dieser Angebote werden als Erweiterung der Themenliste bzw. des Themenbaums dargestellt.

Karte drucken

Ein Mausklick auf dieses Symbol öffnet ein Fenster, über das ein PDF-Dokument der aktuellen Kartenansicht erstellt werden kann. Zusätzlich können Sie  über drei Textfelder, den Titel, eine Beschreibung und den Autor der Karte angeben.

Zur Auswahl stehen zwei Kartenvorlagen; eine mit horizontaler und eine mit vertikaler Seitenausrichtung.

Kartenfenster vergrößern

Der Kartenausschnitt wird auf die gesamte Portalbreite vergrößert.

Kartenfenster verkleinern

Der Kartenausschnitt wird auf die Ausgangsgröße verkleinert.

Nach Auswahl eines Kontextes und Klick auf den 'übernehmen'-Button wird dieser in den Geoeditor geladen.

Umschalten zwischen Legende u. Themenliste

Drop-Down-Liste

Mit Hilfe dieses Bedienelementes kann zwischen der Darstellung der Themenliste und der Legende gewechselt werden. Unterhalb des Umschalters wird das zugehörige Element (Themenliste oder Legende) angezeigt.

Themenliste

Sichtbarkeit von Themen

Die Sichtbarkeit eines Themas wird durch das Kontrollkästchen vor seinem Namen gesteuert. Nur die angehakten (="sichtbaren") Themen werden vom jeweiligen Web Map Service angefordert und so beim Aufbau des Kartenbildes berücksichtigt. Eine Änderung dieser Einstellung führt nicht automatisch zu einer neuen Kartenanfrage, diese Aktion muss vom Nutzer durch Anklicken des Werkzeugs Karte neu zeichnen ausgelöst werden.

Grau hinterlegte Themen können in dem aktuell eingestellten Kartenausschnitt nicht dargestellt werden. Für Themen, die über den Dialog Themenliste erweitern zu einer Karte hinzugeladen worden sind, ist diese Funktion nicht verfügbar. 

Zuordnung von Sachdatenabfragen

Durch diese Auswahlknöpfe (  /  ) wird gesteuert, auf welches Thema sich die Sachdatenabfrage bezieht. Die Sachdatenabfrage kann sich nur auf ein einziges Thema beziehen, die Auswahl eines neuen Themas setzt daher eine zuvor getroffene Auswahl zurück.

Bei Themen, für die keine Sachdatenabfrage möglich ist, wird kein Auswahlknopf angeboten.

Wird ein bei der Sichtbarkeit von Themen auf "nicht-sichtbar" (   ) eingestelltes Thema ausgewählt, wird es dadurch zusätzlich auf sichtbar" (  ) gesetzt.

Umsortieren der Themenliste

Die Einzelthemen werden beim Aufbau des Kartenbildes in der Reihenfolge dargestellt, in der sie in der Themenliste aufgeführt sind. Das oberste Thema in der Themenliste wird dabei im Kartenfenster logisch als letztes "gezeichnet" und überdeckt daher u.U. Inhalte der darunter liegenden Themen. Der Nutzer kann die Einzelthemen aber innerhalb der Themenliste verschieben und die Liste so nach seinen eigenen Wünschen sortieren, um nicht gewünsche überdeckungseffekte zu vermeiden.

Das Einzelthema, das verschoben werden soll, muss zuvor durch Anklicken seiner Bezeichnung ausgewählt werden. Die zugehörige Zeile wird dann gelb hinterlegt (siehe Abb. links). Durch Anklicken der Befehlsschaltflächen oder am unteren Ende der Themenliste wird das ausgewählte Thema innerhalb der Themenliste eine Position nach oben bzw. nach unten geschoben. Ein Umsortieren der Themenliste führt nicht automatisch zu einer neuen Kartenanfrage, diese Aktion muss vom Nutzer durch Anklicken des Werkzeugs Karte neu zeichnen ausgelöst werden. Einzelthemen können auf diese Weise nur innerhalb ihrer Themengruppe verschoben werden. 

Entfernen und Umsortieren von Themengruppen

Für jedes zusätzliche Kartenangebot, das der Nutzer über den Dialog Themenliste erweitern hinzugeladen hat, wird ihm in der Themenliste eine weitere Themengruppe präsentiert. Dabei kann derselbe WMS auch mehrfach (z.B. mit unterschiedlichen Themen) hinzugeladen werden - in diesem Fall werden mehrere Themengruppen mit demselben Namen angezeigt.

Wenn mehrere Themengruppen in der Themenliste vorhanden sind, erscheinen hinter den Bezeichnungen der Themengruppen zusätzliche Befehlsschaltflächen: Mit der Schaltfläche "Themengruppe entfernen" wird eine Themengruppe aus der Themenliste herausgenommen. Nach dieser Aktion erfolgt automatisch ein neuer Kartenaufbau ohne Berücksichtigung der Themen des entfernten WMS.

Mit den Schaltflächen "Themengruppe nach oben schieben" bzw. "Themengruppe nach unten schieben" kann die jeweilige Themengruppe innerhalb der Themenliste um eine Position nach oben bzw. nach unten geschoben werden. Auf diese Weise können ungewollte Überdeckungseffekte auch zwischen Themen aus unterschiedlichen Gruppen verhindert werden. Es werden nur Schaltflächen für diejenigen Richtungen angezeigt, in die eine Verschiebung möglich ist. Nach dem Verschieben einer Themengruppe wird automatisch ein neuer Kartenaufbau mit der geänderten Darstellungsreihenfolge durchgeführt.

Legende 

Legendengrafiken

Im Legendenbereich wird grundsätzlich für jedes sichtbar geschaltete Thema (Sichtbarkeit von Themen) eine Legendengrafik angezeigt, die die kartographische Darstellung des jeweiligen Themas erläutert. Nebenstehend ist eine exemplarische Legendengrafik abgebildet.

Platzhalter

Für zusätzliche Kartenangebote, die der Nutzer über den Dialog Themenliste erweitern hinzugeladen hat, kann nicht immer eine Legende angezeigt werden. Ursache: der Abruf einer Legende von einem WMS ist zwar standardisiert, die tatsächliche Auslieferung einer Legendengrafik ist aber nicht verpflichtend. Daher unterstützen zzt. viele WMS den Abruf von Legenden noch nicht.

In diesen Fällen wird, wie in dem nebenstehenden Beispiel dargestellt, eine Platzhaltergrafik zusammen mit dem Titel (Title) des Kartenthemas (Layer) angezeigt.

Koordinaten-Anzeige

Je nach Position des Mauszeigers, werden die zugehörigen Koordinaten angezeigt.

Timeselect

Timeselect

Das Timeselect-Modul bietet die Möglichkeit zeitabhängige Daten anzufragen und darzustellen.

Ein Klick auf die Schaltfläche des Timeselect-Moduls öffnet ein neues Fenster in dem der Zeitparameter für die Kartenanfrage gesetzt wird.

Die Eingabemaske

In der Eingabemaske des Timeselect-Moduls kann ein Datum und eine Uhrzeit für die Erzeugung des Time-Parameters angegeben werden. Außerdem werden in der Auswahlliste die Layergruppen selektiert, für die der Time-Parameter gelten soll.

Das Datum kann dabei über die Tastatur eingegeben oder mit einem Klick auf das Kalendersymbol über einen Kalender ausgewählt werden. über die Drop-down-Listen kann eine Uhrzeit (Stunden und Minuten) selektiert werden.

Metadatensuche

Mit diesem Modul können Metadaten gesucht werden.

 

Einfache & Detaillierte Suche

Nach dem Starten des Moduls zur Metadatensuche, kann zwischen einfacher und detaillierter Suche unterschieden werden. Bei der einfachen Suche wird lediglich ein Suchstring eingegeben und ausgewertet. Bei der detaillierten Suche kann die Suche z.B. durch Angabe von Themen-, Zeit- und Raumparametern eingeschränkt werden.

 

Layerbasierte Suche

Sie finden diesen Knopf am unteren Ende der Themenliste. Wenn ein Layer durch Klick auf den Layernamen ausgewählt wurde, wechselt das Symbol in Abhängigkeit von der Verfügbarkeit von Metadaten für den selektierten Layer von inaktiv (gräulich) zu aktiv (bläulich).

Ein Klick auf das (aktive) Symbol wird ein neues Fenster mit den Metadaten des gewählten Layers öffnen.

Suchkataloge

Die Suchkataloge bieten eine einfache Möglichkeit, nach raumbezogenen Begriffen zu suchen und das Kartenfenster auf den zugehörigen Ausschnitt zu positionieren. Dazu werden Klapplisten bzw. Eingabefelder verwendet, die je nach ausgewähltem Raumtyp variieren.

Klappliste "Suche über..."

Hier kann ein Katalog mit raumbezogenen Begriffen, nach denen gesucht werden soll, ausgewählt werden.

Digitalisierung 

Das deegree iGeoPortal bietet, neben den gängigen Funktionen zur Navigation auf einer Karte, ein Modul zum Digitalisieren und übertragen eigener Daten mittels WFS-Transaktion. Um das Digitalisierungs-Modul nutzen zu können, muss, neben einigen Anpassungen im Digitalisierungs-Modul selbst, ein WFS und WMS für die digitalisierten Features konfiguriert werden. Weitere Details dazu sind in der Dokumentation zu finden.
